# Sunny Dev Log

Sunny Dev Log is a personal web app created to document debugging notes, reusable code snippets and programming lessons.

This project was built as a visual development journal where I can save problems I solve, the fixes I used and the lessons I want to remember for future projects.

## About the Project

The goal of this project is to turn everyday coding mistakes into useful documentation.

Instead of keeping fixes scattered in chats, notes or memory, Sunny Dev Log organizes them into a small interactive journal with categories, search, pagination and reusable snippets.

## Features

- Dynamic notes rendered from JavaScript
- Notes organized by category
- Folder-style category tabs
- Dynamic note counter
- Search bar for notes
- Notes pagination
- Empty state for categories or searches without results
- Dynamic snippets rendered from a separate JavaScript file
- Expandable snippet search bar
- Snippets pagination
- Responsive layout for smaller screens

This project helped me understand how JavaScript can make a static page feel like a small app.

I learned how to separate data from HTML, render content dynamically and update the interface based on user actions like clicking tabs, typing in a search bar or moving between pages.
